# Edwar Tech - Servicio Técnico Especializado Apple

Página web profesional para **Edwar Tech**, servicio técnico especializado en reparación de equipos Apple ubicado en el Mercado Bambú, Lurín.

## 🛠️ Stack Tecnológico

- **Framework:** Next.js 14 (App Router) + TypeScript
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Iconos:** Lucide React
- **Formularios:** React Hook Form + Zod

## 📦 Instalación

```bash
npm install
npm run dev
```

## 🚀 Build

```bash
npm run build
npm start
```

## 📁 Estructura

```
edwar-tech/
├── app/                    # Páginas (App Router)
├── components/
│   ├── layout/            # Navbar, Footer, WhatsAppButton
│   ├── sections/          # Hero, Services, Terms, Contact, etc.
│   ├── ui/                # Button, Card, Badge, Input
│   └── seo/               # StructuredData, MetaTags
├── lib/                    # Constantes y utilidades
└── public/                 # Imágenes y assets
```

## 📱 Páginas

- `/` — Inicio con Hero, servicios, beneficios y CTA
- `/reparaciones` — Servicios de reparación por dispositivo
- `/condiciones` — Términos y condiciones + formulario
- `/ubicacion` — Ubicación, contacto y mapa
